# Kavya Kanaja 📖

## Overview

Kavya Kanaja is a Kannada literature learning Android application developed using Kotlin and Jetpack Compose during an internship at MindMatrix.

The application is designed to digitally promote Kannada poetry and make literature learning easier, interactive, and beginner-friendly. The app combines modern Android UI design with educational and cultural content to create an engaging literary experience for users.

Kavya Kanaja helps users explore Kannada poems, understand difficult words, read bhavartha (poem explanations), learn about famous Kannada poets, and listen to poems using text-to-speech functionality.

# Project Objectives

- Promote Kannada literature through digital technology
- Help beginners understand Kannada poems easily

- Combine education and culture using technology

---

# Features

## 📜 Poem of the Day
Displays a featured Kannada poem on the home screen.

## 📖 Bhavartha
Provides simplified explanation and meaning of poems.

## 🧠 Difficult Word Meanings
Highlighted Kannada words can be clicked to view English meanings.

## 🔊 Text-to-Speech
Users can listen to poems using speech functionality.

## ❤️ Favorites Section
Users can save favorite poems for later reading.

## 👨‍🎓 Poets Corner
Displays information about famous Kannada poets and Jnanpith award winners.

## 🎨 Premium UI Design
Modern UI with gradients, cards, spacing, and clean layouts designed using Jetpack Compose.

## - Search functionality
Users can search any poets or poems.
